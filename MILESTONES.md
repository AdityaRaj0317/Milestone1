# Project Milestones - Delhi Project

## Overview
This document outlines all milestones for the Delhi single-page website project, including deliverables, timelines, complexity levels, and budget allocation.

---

## Milestone 1: Project Kick-off & Content Definition

**Timeline:** 0.5–1 day  
**Complexity:** Low  
**Budget Allocation:** 15%

### Description
Initial meeting to understand the client's vision for the single page, gather all required text content about Delhi, and identify any specific image requirements. Define the main sections and overall message of the page.

### Expected Deliverable
- Documented content brief
- Agreed-upon page sections and key messages

---

## Milestone 2: Basic UI/UX Wireframing & Style Guide

**Timeline:** 0.5–1.5 days  
**Complexity:** Low-Medium  
**Budget Allocation:** 20%

### Description
Create a low-fidelity wireframe or sketch of the page layout, defining the placement of headings, paragraphs, and image placeholders. Establish a basic color scheme and font style to ensure a cohesive look and feel.

### Expected Deliverable
- Low-fidelity wireframe/mockup
- Basic style guide (colors, fonts)

---

## Milestone 3: Static Page Development (HTML & CSS)

**Timeline:** 1–2 days  
**Complexity:** Medium  
**Budget Allocation:** 30%

### Description
Develop the core single-page structure using HTML and apply fundamental CSS for styling, layout, and basic responsiveness. This milestone focuses on building the framework without integrating final content.

### Expected Deliverable
- Functional static HTML/CSS page with placeholder content
- Layout matching the approved wireframe

---

## Milestone 4: Content Integration & Refinement

**Timeline:** 0.5–1.5 days  
**Complexity:** Medium  
**Budget Allocation:** 20%

### Description
Integrate all approved text content and images into the developed HTML page. Perform necessary CSS adjustments to ensure optimal display and readability of the content across different viewports.

### Expected Deliverable
- Fully populated single-page website with all agreed-upon content
- Complete text and image integration

---

## Milestone 5: Final Review & Deployment

**Timeline:** 0.5–1 day  
**Complexity:** Low-Medium  
**Budget Allocation:** 15%

### Description
Conduct a final review of the complete website with the client, incorporating any last-minute minor feedback or adjustments. Deploy the static website to a specified hosting environment (e.g., GitHub Pages, simple web server) and provide the live URL.

### Expected Deliverable
- Live single-page website URL
- Handover of final project files

---

## Summary

| Milestone | Duration | Complexity | Budget |
